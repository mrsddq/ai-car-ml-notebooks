# Notebook Reproducibility Checklist

Use this repo as a notebook learning archive for AI/car and ML fundamentals.

| Check | Target |
| --- | --- |
| Environment | Record Python version and core packages before rerunning notebooks |
| Data | Keep dataset source links and avoid committing large raw files |
| Outputs | Export key plots or metrics into `docs/` when results are worth sharing |
| Claims | Describe notebook experiments as educational unless rerun with fixed seeds |
| Next step | Convert the strongest notebook into a small script with tests |
